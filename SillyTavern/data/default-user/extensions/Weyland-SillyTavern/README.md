# Weyland Silly Tavern

![](https://github.com/FFFox-ST-Manager/WeylandSillyTavern/blob/main/Welcome.png)

**Weyland SillyTavern** is custom-made layout for Weyland that was designed to prevent plagiarizing of Lucky's characters and the special features of Weyland SillyTavern. 

There is no logging of any data of any kind in any form with this extension. This is simply a UI edit layout relying on the trust of our users.

Please note that if you stumbled upon this somehow, this is not an extension meant for regular SillyTavern users, but only the Weyland SillyTavern experience.

Please enjoy the authentic Weyland experience, and let us know if there is anything we can improve upon.
